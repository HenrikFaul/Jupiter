3. **Auto-tagging / categorization** ‚Äî group images into albums (like the reference screenshot).

## Added

- **`data/media/DeviceDisplayProfile.kt`** ‚Äî true display size (WindowManager on API30+, else
  DisplayMetrics) ‚Üí recommended presets (Match-screen / 1080p / 720p / 480p) that **never upscale**,
  each with a bitrate heuristic.
- **`data/media/MediaCompressor.kt`** ‚Äî images: `BitmapFactory` downscale + JPEG/WEBP re-encode;
  video: **Media3 Transformer** on `Dispatchers.Main` via a `suspendCancellableCoroutine` bridge
  (listener-based completion, `ProgressHolder` polling, cancellation), `Presentation` downscale. Never
  crashes on bad media.
- **`feature/compress/{CompressScreen,CompressViewModel,CompressUiState}.kt`** ‚Äî device line, media
  picker (largest first), preset chips (recommended highlighted), progress, before‚Üíafter result card.
- **`widget/FavoritesWidget.kt` + `FavoritesWidgetReceiver.kt` + `res/xml/favorites_widget_info.xml`**
  ‚Äî a **Glance** home-screen widget listing bookmarked folders/files (via a Hilt `@EntryPoint`); each
  row taps into the app at that path; manifest `<receiver>` added.
- **`data/media/AlbumsSource.kt` + `feature/albums/{AlbumsScreen,AlbumsViewModel,AlbumsUiState}.kt`** ‚Äî
  image **Albums** grouped by MediaStore bucket (Camera/Screenshots/WhatsApp‚Ä¶) ‚Üí album grid ‚Üí images
  grid ‚Üí open; a `suggestTagsFor` per-type auto-tag suggestion helper.

## Remaining risk / rollback

- Medium: Media3 Transformer (video) and Glance (widget) are new deps I couldn't compile locally; CI is
  authoritative and the babysit auto-fixes. Widget re-launch while the app is already running uses
  `SINGLE_TOP` ‚Äî the cold-start deep link is handled; `onNewIntent` refinement is a follow-up. Rollback
  = revert this commit + the deps commit.

## Lessons captured this iteration

- **Glance has two `ColorProvider`s** (unit interface vs color day/night factory) ‚Äî same simple name,
  different packages; qualify to disambiguate.
- **Widget/notification Intents must use the runtime `packageName`, not the namespace** ‚Äî the
  `applicationIdSuffix` (`.debug`) makes a hard-coded package fail to resolve the activity.
- **Media3 Transformer must be built + started on a Looper thread (Main)** and bridged to coroutines
  via its listener. (Sync to `‚Ä¶/lessons-system/lessons_orchestrator.md`.)
