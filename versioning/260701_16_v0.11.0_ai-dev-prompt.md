## Lessons captured this iteration

- **Room + Hilt provisioning**: `@Provides` (DB via `Room.databaseBuilder`, and the DAO) cannot live
  in the same object as `@Binds`; use one `object` module for `@Provides` and a separate `abstract`
  module for `@Binds Impl -> Interface`.
- **`@HiltWorker` needs `@AssistedInject` + the app's `HiltWorkerFactory` `Configuration.Provider`**
  (already present) — mirror the existing worker rather than re-deriving it.
- **Index as a best-effort overlay**: render cached results instantly, then let the authoritative
  live walk reconcile; when the index is empty/disabled the behavior must be byte-identical to before.
  (To be synced into `…/lessons-system/lessons_orchestrator.md` from a machine that can reach it.)
